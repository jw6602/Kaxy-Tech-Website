4. Work Flow for Residential isp hosting

1. Clients bring their ip space to us. Here we require LOA / RIR Object / RPKI / RAdb (Not Necessary).
2. First you need to verify RIR Object / RPKI. If everything is set correctly, send the LOA in email to admin@kaxynetwork.com
3. I will verify everything and submit the LOA to isp for approval.
4. Once the announcement is complete, I will tell you to ask the clients if they want to use our software. If they don't, ask which operating system they want and forward the info to me.
5. Create an invoice to clients and deliver the login details of server to the clients once they finish the payment.

5. How to verify LOA

1. You need to check the prefix very carefully.
2. Format of LOA: This letter serves as authorization for Kaxy Tech LLC to advertise ...

6. How to verify RIR Route Object?

First, we need to know the ip space belongs to which RIR.
RIPE: https://apps.db.ripe.net/db-web-ui/query
ARIN: https://search.arin.net/
AFRINIC: https://afrinic.net/whois
APNIC: https://www.apnic.net/?s=
Simply input one of the ips in the websites above, check if it belongs to each RIR and verify if the route object is set correctly.

7. How to verify RPKI?

Input the prefix (eg. 123.123.123.0/24) into https://rpki-validator.ripe.net/roas

8. RAdb

Input the prefix (eg. 123.123.123.0/24) into https://www.radb.net/

9. Geolocation Update

There are four major ip databases include ipinfo, ip2location, dbip and maxmind.
ipinfo: Email to support@ipinfo.io
dbip: Email to support@db-ip.com
ip2location: Email to support@ip2location.com
maxmind: Email to support@maxmind.com or fill the form in their portal https://support.maxmind.com/geoip-data-correction-request/correct-a-geoip-location/
